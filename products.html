The Bridge

This monitoring and evaluation application is a data-driven, decision-making and school accountability tool. It integrates web technologies into teaching and learning, with a particular focus on tracking learner performance. The Bridge can also be defined as an analytics-based executive information System that covers the whole education system. It provides an early-warning system to the authorities about potential problem areas that require intervention.

Task Tracker

TaskTracker is designed to coordinate teamwork and measure project progress. Its functionality includes enabling IT request approvals, and tracking issues and incidents, software bugs and new product requirements. It further assists with tracking time and effort spent on each daily task and other organisational processes.

IMSMS

This solution is built with municipalities in mind. It automates and streamlines service delivery processes. In addition, it facilitates customer service through logging and tracking of complaints and/or issues reported to the municipality. The average turnaround time it takes to resolve complaints or attend to reported issue provides an indication of organisational efficiency and productivity. The key service delivery areas IMSMS allows the municipality to track include water and sanitation, roads and electricity infrastructure. The tool provides a means to collect information on all municipal service queries. It integrates cloud technologies, enabling seamless deployment. Another IMSMS key feature is a mobile phone application (APP), which allows ratepayers to log queries and to upload incident images.
